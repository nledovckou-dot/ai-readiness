# Готовность ДЦ к ИИ — интерактивный калькулятор

Интерактивная версия плейбука «Внедрение ИИ в дилерской группе» от BACHATA × SPASLEAD.

Один процесс, три оси, понятный балл.

- **Острота × 0.40** — насколько процесс болит
- **Техника × 0.35** — данные, регламенты, интеграции (взвешено по критериям)
- **Команда × 0.25** — готовность людей

Результат: балл от 1 до 5, интерпретация, конкретные следующие шаги.

## Локально

```bash
npm install
npm run dev
```

## Сборка

```bash
NODE_ENV=production npm run build
# артефакты в ./out
```

## Стек
